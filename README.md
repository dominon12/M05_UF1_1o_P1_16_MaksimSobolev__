## Use instructions

- Open terminal

- Clone repository

```bash
git clone https://github.com/dominon12/M05_UF1_1o_P1_16_MaksimSobolev__.git
```

- Cd to repository

```bash
cd M05_UF1_1o_P1_16_MaksimSobolev__
```

- Launch the program

```bash
./main
```

- Enjoy!
